## 🚀 Usage

All components support:
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Dark mode
- ✅ Accessibility (ARIA)
- ✅ Full customization

Import example:
```tsx
import { Button } from "@/components/ui/button"
import { Card } from "@/components/ui/card"
import { Calendar } from "@/components/ui/calendar"

export default function MyComponent() {
  return (
    <Card>
      <Button>Click me</Button>
      <Calendar />
    </Card>
  )
}
```
